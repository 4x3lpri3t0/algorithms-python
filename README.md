# Techniques & Tricks

```python
# Sum of all digits
sum(map(lambda x: int(x)**2, str(n)))

# Count char appearances in string (Linear)
astring = "Hello world!"
astring.count("l") # 3

# Slice string
astring[3:4] # l

# Step (1: all of them, 2: even...) [start:stop:step]
astring[0:5:1] # Hello
astring[0:5:2] # Hlo

# Reverse string
astring[::-1]

# startswith / endswith
astring = "Hello world!"
print(astring.startswith("Hello")) # True
print(astring.endswith("asdfasdfasdf")) # False



```
